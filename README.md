# Django with Nginx Server

This project sets up a basic Django application served with Nginx as a reverse proxy.

## Overview

A lightweight Docker-based setup combining Django (Python web framework) and Nginx (high-performance web server) for production-ready deployment.

## Features

- Django application framework
- Nginx reverse proxy configuration
- Containerized deployment
- Static file serving

## Getting Started

1. Clone the repository
2. Install dependencies
3. Configure Django settings
4. Start the Nginx and Django services

## Architecture

- **Nginx**: Handles incoming HTTP requests and forwards them to Django
- **Django**: Processes application logic and serves dynamic content

## Requirements

- Docker & Docker Compose (optional)
- Python 3.x
- Django
- Nginx
